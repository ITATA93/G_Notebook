# Prompts operativos (Gemini-first)

Estos prompts están diseñados para ser usados por un agente con acceso a Notion (idealmente vía MCP).

## Lista
- `GEMINI_DEPLOY_PROMPT.md`: despliegue completo o idempotente.
- `GEMINI_REPAIR_PROMPT.md`: reconciliar drift con manifest.
- `GEMINI_SEED_PROMPT.md`: cargar seed mínimo `[DEMO]`.
- `REPORT_TEMPLATE.md`: formato estándar de reporte.

## Regla
Siempre pegar al final el YAML completo de `manifests/nos.yaml`.
