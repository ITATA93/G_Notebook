# Convenciones y taxonomía

## Nomenclatura
- Bases de datos: DB_* (mayúsculas).
- Campos: español, legibles.
- Seeds: prefijo `[DEMO]`.

## Estados (Tasks)
- Backlog
- Next
- In Progress
- Waiting
- Done

Regla operativa:
- Si lo harás esta semana: Estado = Next o tiene Fecha asignada.

## Prioridad
- P1 (Crítico)
- P2 (Importante)
- P3 (Normal)

## Esfuerzo
- Rápido
- Profundo
- Baja energía

## IDs clínicos
- CX-AAAA-NNN
